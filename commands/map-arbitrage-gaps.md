# Map Arbitrage Gaps

Run the `arbitrage-audit` skill against a specific industry, market, or business model.

## Steps

1. Ask: "What industry, market, or business model do you want to audit?"
2. Walk through the 3 diagnostic questions from `arbitrage-audit`.
3. For each gap identified, assess closing speed and identify the upstream gap.
4. Synthesize into a gap map with positioning recommendations.
5. Save as `gap-map-<industry-slug>.md` in the current directory.
